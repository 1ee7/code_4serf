---------------------------

raspi-code-v1.2
---------------------------
  串口1，波特率为115200；功能有：透传mavlink数据，检测usb0的热插，桥接eth1<->usb0 。usb0 热插检测程序（方法）改变为sock--ioctl方式,每隔一段时间主动查询，而不是之前的被动的触发。

raspi-code-v1.8
-----------------------------
